Introduction

I'm very excited to announce this as the first solid and stable release of karma, the Oracle Database Monitoring software. It's hard to believe that it's been in development now for an entire year, but looking through the changelog, I released the first version around July 7, 1999.

The port of Oracle to Linux is no longer new, and some sites are using it in production environments. Karma is not a Linux only tool, but it's my development platform, and it was Oracle's port to Linux that made the project possible.

I've gotten much of my initial inspiration for Karma from Sean McGuire's Big Brother Network monitoring tool which has evolved substantially in the last year as well. Take a look: Big Brother Home and Big Brother Demo.
