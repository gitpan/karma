Introduction

With the relatively recent port of Oracle to Linux, I've been a bit motivated to build an Oracle tool that I would personally use. I've gotten much of my initial inspiration for Karma from Sean McGuire's Big Brother Network monitoring tool. Take a look: Big Brother Home and Big Brother Demo.
